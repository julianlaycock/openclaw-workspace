## Important Technical Notes
- Use SSE (Server-Sent Events): `Content-Type: text/event-stream`, each event: `data: <json>\n\n`
- Stream from Next.js API route using `ReadableStream` and `TransformStream`
- Frontend: use `EventSource` or `fetch` with streaming to consume SSE
- The GLEIF API is public, no auth needed
- Use `packages/open-annex-iv/dist/index.js` for XML generation (it's compiled)
- libxmljs2 is in packages/open-annex-iv/node_modules — import from there for validation
- Write all files with LF line endings (not CRLF) to avoid Railway build failures
