Common questions.

What frameworks does Caelith cover?+
AIFMD II, KAGB (§1(19)), ELTIF 2.0, RAIF, SIF, MiFID II. 13 rules across 6 frameworks. Annex IV validated against ESMA XSD Rev 6.
Can I try without signing up?+
Yes. The Readiness Check and Compliance Copilot are available instantly — no account needed. The full dashboard demo is available on desktop.
Do you have an API?+
Full REST API with OpenAPI/Swagger docs at /api/docs. Versioned routes, API key auth. Available on Professional and Enterprise. Core library (open-annex-iv) is open source.
What are EMT/EET/EPT?+
European MiFID Template, ESG Template, and Cost Template — standardized data exchange formats. Caelith auto-generates them with 43+ fields pre-filled.
When does AIFMD II take effect?+
April 16, 2026. Fund managers must be fully compliant by this date.
Is my data secure?+
EU-hosted (Frankfurt), end-to-end encryption, GDPR compliant. Data never leaves the EU. Enterprise can opt for on-premise.
