# Installing Caelith as a Desktop/Mobile App

Caelith can be installed as a Progressive Web App (PWA), giving you a native app experience with offline-capable UI, taskbar/dock integration, and full-screen mode.

## Desktop (Chrome / Edge)

1. Navigate to [caelith.tech](https://caelith.tech) in Chrome or Edge
2. Look for the **install icon** (⊕) in the address bar, OR
3. Click the three-dot menu → **"Install Caelith"**
4. The app will open in its own window with the Caelith icon in your taskbar/dock

## Desktop (Safari)

1. Navigate to [caelith.tech](https://caelith.tech) in Safari
2. File → **"Add to Dock"** (macOS Sonoma+)

## Mobile (iOS)

1. Open [caelith.tech](https://caelith.tech) in Safari
2. Tap the **Share** button (square with arrow)
3. Scroll down and tap **"Add to Home Screen"**
4. Tap **"Add"** — Caelith will appear as an app icon

## Mobile (Android)

1. Open [caelith.tech](https://caelith.tech) in Chrome
2. Tap the three-dot menu
3. Tap **"Add to Home screen"** or **"Install app"**
4. Confirm — Caelith will appear in your app drawer

## What You Get

- **Standalone window** — no browser chrome, feels native
- **Taskbar/dock icon** — quick access like any other app
- **Dark theme** — status bar matches Caelith's Midnight Signal theme (#0A0E1A)
- **Fast launch** — cached assets load instantly

## Technical Notes

- PWA manifest: `/manifest.json`
- The app requires an internet connection for data (compliance data is server-side)
- Updates are automatic — refresh or reopen to get the latest version
