Common questions.

What frameworks does Caelith cover?+
AIFMD II, KAGB (§1(19)), ELTIF 2.0, RAIF, SIF, MiFID II. 13 rules across 6 frameworks. Annex IV validated against ESMA XSD Rev 6.
Can I try without signing up?+
Yes. The Readiness Check and Compliance Copilot are available instantly. The full dashboard demo is best on desktop or tablet.
Do you have an API?+
Full REST API with OpenAPI/Swagger docs. Versioned routes, API key auth. Available on Professional and Enterprise. Core library is open source on npm.
What are EMT/EET/EPT?+
European MiFID Template, ESG Template, and Cost Template — standardized data exchange formats. Caelith auto-generates them with 43+ fields.
When does AIFMD II take effect?+
April 16, 2026. Fund managers must be fully compliant by this date.
Is my data secure?+
EU-hosted (Frankfurt), end-to-end encryption, GDPR compliant. Enterprise can opt for on-premise deployment.
