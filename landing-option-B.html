Frequently asked questions

Everything you need to know about deploying Caelith agents.

How does the agent file to BaFin?+
The BaFin Filing Agent reads your fund data, maps it to the ESMA Annex IV taxonomy (147 fields), generates XSD-validated XML, authenticates with BaFin's MVP Portal using certificate-based mTLS, uploads the filing, and captures a confirmation receipt with a SHA-256 hash. The entire process runs autonomously overnight — you wake up to a confirmation email and an audit trail.
Is the XML ESMA-validated?+
Yes, every XML file is validated against ESMA's official Annex IV XSD schema (Rev 6, AIFMD_DATMAN_V1.2.xsd) before submission. The validation is zero-tolerance: if any error is detected, the filing is paused and you're notified immediately. We've maintained a 100% XSD pass rate across all filings.
Where is my data stored?+
All data is processed and stored on EU-hosted infrastructure (Railway Amsterdam, eu-west region). We are GDPR-compliant by design — no data leaves the European Union. We provide a Data Processing Agreement (DPA) to every customer, and our audit trails are available for regulatory inspection at any time.
Can I try it before buying?+
Absolutely. We offer a free demo dashboard where you can explore the platform with sample fund data. You can see exactly how the agents work, review a sample Annex IV filing, and test the Compliance Copilot. No credit card required — just book a demo and we'll set you up.
