Quick Pick

D1 Clean default. Technical + warm dot. Works everywhere.
D2 Same but bolder. More presence in headers/banners. Might feel heavy at small sizes.
D3 Softer. Mixed case + underline = more approachable. Good if "all-caps" feels too aggressive for compliance buyers.
D4 D1 with premium gradient dot. Most visually striking on dark. The glow is subtle.

My pick: D1 for versatility, D4 if you want that extra polish.
