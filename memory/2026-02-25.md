### Key Learnings
- **Railway `file:` deps are unreliable**: Never use `file:packages/...` in package.json for Railway deploys. Inline the code or publish to npm first.
- **tsx cache at `%TEMP%\tsx-julian`**: Must clear when modules change. Local sandbox junction adds confusion.
- **`process.exit()` in scripts**: Never use in scripts that might be imported by other processes. Use throw instead, or run as child process.
- **Copilot language**: Must pass locale through ALL layers (React context → API client type → route type → service interface → system prompt). One missing layer = silent failure.
- **Railway "Completed" vs "Online"**: "Completed" = process exited cleanly (exit 0). Server must stay running to show "Online".
