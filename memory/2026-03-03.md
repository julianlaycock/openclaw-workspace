### Key Technical Lesson
- **Railway migration path**: `scripts/migrate.ts` uses `path.join(__dirname, '../migrations')` → reads ROOT `migrations/` folder only. `src/migrations/` is dev-only (server.ts inline runner, dev mode). Always put Railway-targeted migrations in root `migrations/` OR use startup functions like `ensureDemoUsers()`.
- **Template literal escaping**: `\'` in TS template literal = `'` in output (no escape). Use `\\'` to get `\'` in rendered JS.
