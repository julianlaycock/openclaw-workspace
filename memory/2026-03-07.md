## Week 1 Execution: BaFin Filing Agent + Pipeline API

### MAJOR DISCOVERY: BaFin has a SOAP Web Service
**Not just a portal!** BaFin provides programmatic SOAP API access for AIFMD filings:
- **Production**: `https://portal.mvp.bafin.de:444/services/ws/aifmd`
- **Test**: `https://portal.mvp.bafin.de:444/services/ws/t_aifmd`
- **Auth**: WS-Security UsernameToken (`username#entityId` + password)
- **Journal**: `https://portal.mvp.bafin.de:444/services/ws/protokoll` (status tracking)
- **No MTOM**, no GZIP, no CAPTCHA — just HTTPS + SOAP + WS-Security
- **Response**: Synchronous `meldungsId` (report ID)

This is massive — means the filing agent doesn't need headless browser automation. The existing Playwright agent becomes a fallback, SOAP is primary.
