3. Architecture Layers

🖥️ Frontend

  • Next.js App Router architecture
  • 37+ React components
  • Tailwind CSS with dark/light themes
  • Client-side form validation
  • Responsive dashboard UI

⚡ API Layer

  • Express with versioned routes (/api/v1/)
  • OpenAPI/Swagger documentation
  • API key authentication for external access
  • Public API endpoints for integrations

🔧 Services

  • 60+ service modules
  • Compliance engine, sanctions screening
  • LEI lookup via GLEIF API
  • Annex IV XML generation & XSD validation
  • Audit trail & integrity chain

💾 Data Layer

  • 12 repository modules
  • 73 database migrations
  • Row-Level Security (RLS) for multi-tenancy
  • 100+ indexes for query optimization
  • Soft deletes on key entities

🔒 Security

  • 3-tier rate limiting (auth, API, export)
  • CSP headers, CSRF protection
  • Input sanitization middleware
  • bcrypt password hashing
  • JWT + httpOnly cookies
  • Login attempt tracking & lockout

📊 Monitoring

  • Sentry error tracking
  • Plausible analytics (privacy-first)
  • Health check endpoints (/api/health/ready)
  • Audit trail with integrity chain

  • Tenant isolation indexes on every tenant-scoped table
  • Composite indexes for common query patterns
