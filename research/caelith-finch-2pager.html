Architecture
How it works.

AIFMD II compliance follows deterministic rules: ESMA defines the schema, KAGB defines classification, BaFin defines the submission format. There is no ambiguity in the process — only in executing it correctly across hundreds of fields and multiple jurisdictions. Caelith encodes those rules as typed validation logic, not LLM reasoning. GLEIF, EU/UN sanctions databases, and ESMA's XSD return verified facts. The agent handles execution; the compliance officer approves at two explicit checkpoints — one to generate the XML, one to submit to the NCA portal.
