Key changes from current:
  • 3 sections → 4 lighter sections: Dashboard (pinned), Manage, Compliance, Monitor
  • Entity grouping: "Funds & Assets" = Funds + Holdings + Transfers. "Investors" = Directory + Onboarding + Screening
  • Active bar indicator (left edge) — more refined than full row highlight
  • Badges surface urgency without requiring a click (3 pending decisions, 2 upcoming deadlines)
  • Section renamed: "Oversight" → "Monitor" (shorter, more active verb)
  • Auto-collapse: only active section stays open — sidebar never exceeds viewport
  • Sub-items use dot + indent — light, scannable, doesn't fight the parent items

Key changes from current:
  • Quick-action strip — the 3 most common daily actions (add investor, new transfer, generate report) always visible, zero navigation
  • Tree-line sub-items — left border connects child pages to parent, visually clearer hierarchy than dots
  • Sub-items always visible (no collapse) — flatter, faster, less clicking. Works because there are only 2-3 per group
  • Item count badges on section headers — shows depth without expanding
  • Calendar + Audit Log ungrouped at bottom — they're utility pages, not a "section"
  • Search bar styled as input — looks clickable/inviting, not hidden as a text link
  • Portfolio + Compliance = only 2 real sections. Cleaner mental model.
